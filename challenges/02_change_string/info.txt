DESCRIPTION: Change the string value from "hello world" to "goodbye"
OPTIMAL: 11
SOLUTION_A: ci"goodbye<Esc>  —  change inside quotes, type new value (c i " = 3 keys + 7 chars + Esc)
SOLUTION_B: f"ci"goodbye<Esc>  —  find quote first, then ci" (2 extra keys, unnecessary)
NOTE: ci" works from anywhere on the line — it searches forward for the nearest ". No need to navigate to the quote first.
