DESCRIPTION: Clear the string value, leaving empty quotes — "active" becomes ""
OPTIMAL: 3
SOLUTION_A: di"  —  delete inside quotes (3 keys; fastest here)
SOLUTION_B: ci"<Esc>  —  change inside quotes to nothing, then exit insert (4 keys; same result)
NOTE: di" is one key fewer than ci"<Esc> because there is no insert mode. Try both.
